## 🌱 Git
### branch
> `Pull Request` 와 `Code Review` 가 이루어진 후, 병합을 진행해야 한다.
- `main` : 배포 브랜치
- `front_end` : Front-End 개발 브랜치
- `back_end` : Back-End 개발 브랜치
- `각개인 브랜치/기능 이름/#이슈번호` : FE 기능 개발 브랜치 / BE 기능 개발 브랜치

### Git 저장소 이용법 
로컬에서 작업 후 개인 branch Push
1. `git checkout 브랜치`
2. `git checkout "각개인이 사용할 브랜치`
2. `git add 파일/디렉토리 경로`
3. `git commit -m "message"`
4. `git pull origin "FE/BE브랜치"`
5. `git push`
6. `git pull request 생성시 merge진행확인하기 `
